# Interactive Election Map

An interactable map where the user can see office positions for the federal, state, and county level, intended
to make it easier to get informed about your state and local elections.

Users can see who is the current holder of that office and who is in the running against them in the current election.

### Features
* Link to see campaign website (if present)
* Search function to OpenSecrets.org to see campaign fundraising
* Filter function to sort by party
